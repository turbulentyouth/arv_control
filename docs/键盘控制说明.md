# 键盘控制水下机器人说明

## 前置条件

1. 启动程序后，在设置面板中确保 **"键盘控制"** 开关已开启（默认开启）
2. 程序窗口获得键盘焦点（点击程序窗口即可）
3. MAVLink 已连接到飞控（程序通过 `udpin:0.0.0.0:14550` 监听连接）

## 运动控制

键位布局基于双手操作设计，左手控制水平和转向，右手控制油门和升降：

| 按键 | 功能 | 说明 |
|------|------|------|
| **W** | 前进 | 机器人向前移动 |
| **S** | 后退 | 机器人向后移动 |
| **A** | 左转（偏航） | 机器人绕垂直轴逆时针旋转 |
| **D** | 右转（偏航） | 机器人绕垂直轴顺时针旋转 |
| **J** | 左移 | 机器人水平向左平移 |
| **L** | 右移 | 机器人水平向右平移 |
| **I** | 上升 | 增加油门，机器人上浮 |
| **K** | 下降 | 减少油门，机器人下潜 |

> **设计思路**：左手 WASD 控制平面运动（前进/后退/转向），右手 IJKL 控制垂直和横向运动（升降/横移）。

## 油门大小调节

- 在设置面板中拖动 **"油门上限"** 滑块，范围 0% ~ 100%，默认值 20%
- 油门上限决定了按键时遥控信号偏离中位的幅度
- 计算公式：`信号值 = 1500 ± (油门上限% × 4)`
- 例如：20% 时幅度为 ±80，W 键输出信号 1580，S 键输出信号 1420
- 建议初次操作时使用较低油门值（10%~20%），熟悉后再逐步增大

## 功能按键

| 按键 | 功能 | 说明 |
|------|------|------|
| **空格键** | 解锁/锁定 | 切换机器人的锁定状态。锁定状态下电机不转，解锁后电机响应控制指令 |
| **数字 1** | MANUAL 模式 | 切换到手动模式（custom_mode = 19），操控最直接 |
| **数字 2** | ALT_HOLD 模式 | 切换到定高模式（custom_mode = 2），机器人自动保持当前深度 |

## 爪子控制

爪子控制通过通道 7 发送 PWM 信号，**不受电机锁定状态影响**——即使机器人在锁定状态下，爪子依然可以操作。

| 按键 | 功能 | PWM 值 | 说明 |
|------|------|--------|------|
| **-** （减号/连字符） | 爪子张开 | 1900 | 按住时爪子持续张开 |
| **=** （等号） | 爪子闭合 | 1100 | 按住时爪子持续闭合 |

> 松开按键后，通道 7 恢复到 65535（无 override），爪子保持当前位置。两个按键同时按下时，张开优先。

## RC 通道映射

键盘操作最终通过 MAVLink `RC_CHANNELS_OVERRIDE` 消息发送给飞控，映射关系如下：

| 通道 | 功能 | 控制键 | 中位值 | 范围 |
|------|------|--------|--------|------|
| chan1 | 俯仰 (Pitch) | 无 | 65535 | 禁用 |
| chan2 | 横滚 (Roll) | 无 | 65535 | 禁用 |
| chan3 | 油门 (Throttle) | I / K | 1500 | 1500 ± 油门幅度 |
| chan4 | 偏航 (Yaw) | A / D | 1500 | 1500 ± 油门幅度 |
| chan5 | 前进/后退 (Forward) | W / S | 1500 | 1500 ± 油门幅度 |
| chan6 | 横向 (Lateral) | J / L | 1500 | 1500 ± 油门幅度 |
| chan7 | 爪子 (Gripper) | - / = | 65535 | 1100 (合) / 1900 (开) |
| chan8 | 保留 | 无 | 65535 | 禁用 |

## 安全提示

1. **首次操作**：建议在开阔水域、油门上限设为最低（10%）时进行
2. **键盘控制关闭**：在设置中关闭键盘控制开关后，所有运动按键状态会被重置为零，防止误操作
3. **失控保护**：若 MAVLink 连接断开，程序会自动尝试每 3 秒重连一次，期间不会发送控制指令
4. **控制频率**：程序以 10Hz 频率持续发送 RC override 指令，松开按键后对应通道立即回到中位值（1500）
5. **锁定优先**：入水前务必确认机器人为锁定状态；操作结束后按空格键及时锁定

## 画面显示信息

程序从飞控心跳/遥测数据中解析并显示以下状态：

- **模式**：MANUAL / ALT_HOLD / STABILIZE
- **锁定状态**：锁定 / 解锁
- **深度**：来自 GLOBAL_POSITION_INT 的相对高度（负值表示水下深度）
- **姿态**：俯仰角 (Pitch)、横滚角 (Roll)
- **油门百分比**：来自 VFR_HUD
- **电量**：来自 SYS_STATUS
- **航向**：罗盘显示，来自 VFR_HUD
